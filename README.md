### Blog App
This is a mobile web application where you can manage your budget: you have a list of transactions associated with a category, so that you can see how much money you spent and on what.

### Built with
- Ruby on Rails
- VS Code
- Postgresql
### Pre-requsite
- Ruby
- VS Code
- Postgresql
- Rails
